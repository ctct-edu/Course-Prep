# CTC教育サービス

## Microsoft関連 コース ガイド



### ■はじめに

CTC教育サービスが提供する「**Microsoft関連コース**」をご受講いただき、誠にありがとうございます。

コース受講前に事前準備が必要となります。

「**コース一覧**」より、ご受講予定コース名をクリックし、内容をご確認ください。
